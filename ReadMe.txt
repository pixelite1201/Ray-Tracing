INSTRUCTIONS:

To compile:

make

To run:

./raytracer input.scn


Output:

ppm image will be generated

Description:

Modifications can be made in input.scn file as described in scene.scn file to output a different image.
Details about project are available here https://github.com/pixelite1201/pixelite1201.github.io/blob/master/ray_tracer_1.md
